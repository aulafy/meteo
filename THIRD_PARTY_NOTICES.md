# Avisos de terceros

El código propio de METEO se publica bajo MIT. Sus dependencias conservan sus licencias originales; por tanto, no todo el árbol de dependencias es MIT.

Dependencias directas relevantes:

- React, Supabase JS, Turf, Zod y Fast XML Parser: MIT.
- MapLibre GL JS: BSD-3-Clause.
- toGeoJSON: BSD-2-Clause.
- Lucide React, TopoJSON Client y World Atlas: ISC.
- web-push: MPL-2.0.

METEO toma como referencia patrones de [GeoLibre](https://github.com/opengeos/GeoLibre) y [MeteoFlow](https://github.com/aulafy/meteoflow), cuyos avisos y licencias deben conservarse cuando se reutilice código de esos proyectos.

`package-lock.json` identifica el árbol exacto instalado. Los textos completos de licencia se distribuyen en los paquetes de origen correspondientes y prevalecen sobre este resumen.
